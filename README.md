ESCOLA ESTADUAL PLÍNIO RODRIGUES DE MORAIS
===================================

Site institucional desenvolvido para a Escola Estadual Plínio Rodrigues de Morais,
com foco em organização, clareza das informações e layout moderno, mantendo uma
identidade visual sóbria e institucional.

OBJETIVO DO PROJETO
-------------------
Criar um site institucional que:
- Apresente a escola de forma clara e organizada
- Facilite o acesso a serviços importantes (Sala do Futuro, Boletim, RA)
- Disponibilize informações institucionais e de contato
- Utilize boas práticas de HTML, CSS e organização de código

TECNOLOGIAS UTILIZADAS
---------------------
- HTML5
- CSS3
- SVG inline (controle de cor via CSS com currentColor)
- Google Fonts (Montserrat)
- Git e GitHub

SEÇÕES DO SITE
--------------
- Header: logotipo, menu de navegação, dropdown de contato e login
- Hero: vídeo institucional com overlay e slogan
- Serviços: cards com acesso rápido aos serviços da escola
- Sobre Nós: histórico e informações institucionais
- Contato: WhatsApp, Instagram, Facebook e localização no Google Maps
- Footer: créditos e direitos autorais

DESTAQUES TÉCNICOS
------------------
- SVGs padronizados com currentColor
- Wrapper fixo para ícones garantindo alinhamento consistente
- Layout responsivo usando Flexbox e Grid
- Código organizado e de fácil manutenção

RESPONSIVIDADE
---------------
O site é totalmente responsivo, adaptando-se a:
- Desktop
- Tablets
- Smartphones

COMO EXECUTAR O PROJETO
----------------------
1. Abra a pasta do projeto
2. Clique duas vezes no arquivo index.html
3. O site será aberto no navegador padrão
